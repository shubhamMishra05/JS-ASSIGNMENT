Explain about Console Function.

  1. console.log()

    2. Mainly used to log(print) the output to the console. We can put any type inside the log(), be it a string, array, boolean etc.

  2. console.error()

    3. Used to print error message to the console. Useful in testing of code. By default the error message will be highlighted with red color.

  3. console.warn()

    4. Used to print warning message to the console. By default the warning message will be highlighted with yellow color.

  4. console.clear()

    5. Used to clear the console.

  5. console.table()

    6. This method allows us to generate a table inside a console. The input must be an array or an object which will be shown as a table.

  6. custom console log

    7. User can add Styling to the console logs in order to make logs Custom.
