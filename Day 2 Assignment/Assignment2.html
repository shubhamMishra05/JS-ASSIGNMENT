Difference between var , let and const

var
Scope can be global and function,But it's scope can not be block.variables can be reassigned.it also be re-declared.
let
Scope can be function and block only it's scope can not be global.and it also be reassigned but not be re-declared
const
Scope can be function and block only it's scope can not be global.but it can not be reassigned or re-declared.It is necessory to initialize the const at the time of declaration.
